III. 리액트 앱 실행하기 종료하기

1. 명령 프롬프트에서 프로젝트 폴더로 이동한 다음 > npm start로 앱을 실행한다
2. Terminal을 이용 할 때는 경로를 주의해야 한다.
3. 웹 브라우저에 리액트 로고가 나타나면 정상적으로 프로젝트가 생성된 것이다.
4. App 실행되면 웹브라우저에 다음 화면이 나타난다.
5. App을 종료하려면 프롬프트에서 Ctrl+c를 누른다.
